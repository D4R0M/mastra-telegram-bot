## SM-2 Algorithm

* Initialize EF=2.5, reps=0, interval=0.
* On grade `q` (0â€“5):

  * If `q < 3`:

    * reps=0, interval=1d, due=now+1d
  * Else:

    * EF = max(1.3, EF + (0.1 - (5-q)\*(0.08 + (5-q)\*0.02)))
    * If reps==0: interval=1d
    * If reps==1: interval=6d
    * Else: interval=round(interval\*EF)
    * reps+=1; due=now+interval
